## 4. 아이디어 제안
### Good Data : 데이터 라벨링으로 쉽게 하는 기부
- 인공지능 연구에 필요한 데이터 라벨링 작업을 통해 간편하고 재미있는 기부문화 선도
- 인공지능 연구에 있어서, 데이터에 의미를 부여해주는 데이터 라벨링 작업이 대규모로 요구됨
- 키오스크를 통해 간단한 작업만 진행한다면 자신의 돈을 사용하지 않고도 기부를 할 수 있는 기대효과
